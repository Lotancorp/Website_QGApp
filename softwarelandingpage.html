Quartz Toolkit — RF Online Modding Utility

A cohesive suite of tools designed to streamline RF Online content creation — from EFF/SPR handling to icon and particle indexing. Built for modders, creators and server maintainers who need speed, accuracy and batch-processing power.

Core Features

Index Number SPR

Scan and generate reliable index maps for SPR collections, ensuring consistent referencing for skins and animations.

Join & Extract Icon

Join multiple icon sources into single sheets and extract icons with accurate cropping for game-ready assets.

Custom Icon Creator (Add-On)

Create icons with customizable backgrounds via an Add-On module — batch create or tweak per-item assets quickly.

EFF ⇄ Excel Converter

Convert EFF files into Excel for easier editing, and reconvert modified spreadsheets back into EFF with structural integrity.

Chef RF Folder Generator

Automated folder configuration tailored for RF Online server file structures to speed up deployment and testing.

Particle.ini Resource Indexer

Index and validate particle resource numbers to check availability and avoid collisions in particle lists.

EFF Structorian

A structural inspector and fixer for EFF files — validates format, repairs common inconsistencies, and reports issues.

Mass Entity & SPT Color Editor

Bulk edit entity definitions and SPT color tables with preview and batch-apply operations for massive content updates.

DDS DXT Type Converter (Add-On)

Convert DDS textures between DXT compression types via an Add-On for compatibility and performance testing.

DDS DXT Type Checker (Add-On)

Quickly scan and report DDS compression types to ensure assets meet server/client expectations.
